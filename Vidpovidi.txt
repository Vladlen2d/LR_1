Що таке GIT? - розподілена система керування версіями файлів та спільної роботи
В якому статусі можуть перебувати файли при роботі з git - перевірка 
Для чого використовується команда add ? - Ви можете думати про це як про чисте полотно, яке містить зміни, які ви хотіли б внести
Для чого використовується команда status ? - Вона повертає інформацію про поточний стан репозиторія:
Для чого використовується команда commit ? - Коміт відображає стан нашого репозиторія у момент, коли він був зроблений
Як переглянути список комітів через комнаду git log
Для чого використовується команда checkout ? Для переходу між гілками
